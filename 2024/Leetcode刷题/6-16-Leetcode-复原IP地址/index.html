# 算法思想

  • 基本还是分割问题，但是和前面一个分割回文串的区别就是，本题中强制要求把字符串分为 4 串，所以最后需要进行限制 (可以有不同的方法)
  • 另外就是其实这一个题和分割回文串的题目差不多，只要改变判断条件和终止条件就可以了，注意如果进行判断
  • 另外的思路:
    • 第一种方式：还是利用一个集合来记录分割之后的串，但是这一种方法得到的就是集合，最后还需要把集合转换为字符串
    • 第二种方法：就是实时添加 中间的分割符号，这样的话，可以使用一个变量用于记录分割符号的个数，当分割符号的个数为 3 时就可以退出并且判断最后一个分割得到的字符串是否满足要求了，但是此时的一个细节就是动态添加。之后，下一次递归需要在逗点的下一个位置开始进行
